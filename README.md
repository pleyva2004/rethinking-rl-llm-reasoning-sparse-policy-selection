## The paper

**Title:** Rethinking RL for LLM Reasoning: It's Sparse Policy Selection, Not Capability Learning
**Authors:** Ömer Faruk Akgül, Rajgopal Kannan, Willie Neiswanger, Viktor Prasanna
**Venue:** arXiv preprint, May 2026
**Official code:** https://github.com/farukakgul/ReasonMaxxer

**Headline claim.** RL post-training for math reasoning doesn't teach new strategies — it reranks ~2% of tokens at high-entropy decision points toward alternatives the base model already had in its top-5. An offline contrastive fine-tune gated on base-model entropy (ReasonMaxxer) matches or exceeds full RL across 3 model families × 6 scales × 6 benchmarks at ~1000× lower training cost.

## Run the sandbox

```bash
cd sandbox
pip install -r requirements.txt
python experiment.py
```

Loads `Qwen2.5-0.5B-Instruct` (~1GB), greedy-decodes a math chain-of-thought, computes per-token predictive entropy from the model's own logits, and prints the trace annotated by entropy with positions above $\tau = 1.4$ marked as decision points.

The probe asks: *do the high-entropy positions in a small base model cluster at semantically meaningful branching tokens, as the paper predicts for larger models?* See [`sandbox/README.md`](./sandbox/README.md) for the full experiment design and falsification criteria.
